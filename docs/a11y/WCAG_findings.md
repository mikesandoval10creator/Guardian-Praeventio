> Companion to `A11Y_AUDIT.md`. Every finding cites a real
> `file:line` reference. Severity uses the SEV1/SEV2/SEV3 rubric:
>
> - SEV1 (critical) — blocks usage with assistive tech (screen readers, switch control, voice control)
> - SEV2 (medium)   — degrades experience but workaround exists
> - SEV3 (low)      — cosmetic / nice-to-have / non-text contrast at borderline thresholds
>
> Status: `pass` / `partial` / `fail` / `mitigated` (post-fix).
