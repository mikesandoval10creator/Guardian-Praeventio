# Guardian Praeventio — STRIDE Findings

Companion to [`THREAT_MODEL.md`](./THREAT_MODEL.md). Each row points to a
real file:line reference. Status is one of:

- `mitigated` — implemented control verified by test or by direct read.
- `partial` — control exists but has a known gap; the gap is the open work.
- `open` — no control today; tracked for a future bucket.

Severity:
- `[H]` critical (immediate impact, broad blast radius)
- `[M]` medium (requires precondition or limited scope)
- `[L]` low (defense-in-depth)

Categories: S=Spoofing, T=Tampering, R=Repudiation, I=Information disclosure,
D=Denial of service, E=Elevation of privilege.
